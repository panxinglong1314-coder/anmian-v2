# CBT-I 失眠认知行为疗法 — 完整操作手册 v2.0
# 基于 2023-2025 最新临床指南

## 概述

CBT-I（Cognitive Behavioral Therapy for Insomnia）是失眠的一线非药物治疗方案。
**2025年**，AASM（美国睡眠医学会）、中华医学会等多机构均将 CBT-I 列为慢性失眠的**首选（first-line）治疗**。

本手册 v2.0 更新要点（相比旧版）：
- 新增 **矛盾意向法**（Paradoxical Intention）—— 2025 中国指南 A 级推荐
- 新增 **第三波 CBT 组件**（正念 + ACT）—— 2024-2025 dCBT-I 系统评价支持
- 更新各组件**证据等级**（1A / 2A / 2B）
- 新增 **dCBT-I 数字化实施框架**（参考 SleepioRx 协议）
- 新增 **失眠亚型适配**（入睡型 / 维持型 / 混合型 / 社交反刍型）
- 更新 **会话状态机**，加入矛盾意向阶段

---

## 证据等级说明（2025 指南）

| 等级 | 含义 | 推荐组件 |
|------|------|---------|
| **1A** | 强推荐，高质量证据 | 刺激控制、睡眠限制、放松训练（PMR/呼吸）、矛盾意向 |
| **1B** | 强推荐，中等质量证据 | — |
| **2A** | 推荐，高质量证据 | 睡眠卫生教育（作为综合方案一部分）、认知重构、正念身体扫描 |
| **2B** | 推荐，中等质量证据 | 慈心冥想、睡眠努力 reduction（新兴） |

> **注**：单独使用睡眠卫生教育效果有限，需与行为干预（刺激控制/睡眠限制）联合使用。

## 核心模块（按执行顺序）

### 模块 0：失眠亚型识别（Insomnia Subtype Recognition）
**目标**：快速判断用户属于哪种失眠类型，选择最有效的技术组合

| 亚型 | 特征 | 首选技术（优先级排序） |
|------|------|----------------------|
| **入睡困难型** | 躺下后 ≥ 20 分钟无法入睡 | ①矛盾意向 ②睡眠限制（轻量） ③4-7-8 呼吸 |
| **睡眠维持型** | 夜间频繁醒来或凌晨早醒 | ①睡眠限制 ②刺激控制 ③正念开放觉察 |
| **混合型** | 入睡 + 维持都困难 | ①睡眠限制 ②认知重构 ③PMR+正念组合 |
| **社交反刍型** | 睡前反复思考人际关系 | ①慈心冥想 ②担忧外化 ③开放觉察 |

**识别问题**（每次会话开始问 1 个）：
> "你今晚睡不着，主要是哪一种情况？①很难入睡 ②半夜会醒 ③脑子里一直在想事情 ④没什么特别，就是睡不着"

### 模块 1：睡眠评估（Sleep Assessment）
**目标**：快速了解用户今晚的睡眠准备状态

**执行时机**：每次会话开始时

**评估维度**：
1. **入睡困难程度**（0-10）："今晚躺在床上到睡着，大概需要多久？"
2. **今晚的主要担忧**：开放性问题，"今晚最让你放不下的是什么？"
3. **最近睡眠状况**：简要了解过去 3 天睡眠
4. **今晚的可控因素**：饮酒/咖啡因/白天小睡/运动情况

**AI 输出格式**：
```
【今晚评估】
入睡困难度：{n}/10
主要担忧：{描述}
失眠亚型：{类型}
建议优先处理：{担忧/CBT技术/直接放松}
会话模式：{轻量版/标准版/强化版}
```

### 模块 2：担忧处理（Worry Processing）
**目标**：将担忧从脑中转移到外部，减轻睡前认知激活

**理论基础**：
- "担忧写下来"技术：语言化担忧可降低其情绪强度
- 控制在 2 轮内完成，避免反刍
- 不追问细节，不给建议方案

**标准流程**：
1. **确认感受**（1句）："听起来你为[担忧来源]感到不安"
2. **外化担忧**（1句）："来，我们先把它写下来——说出来就好，不用想清楚"
3. **记录承诺**（1句）："这件事我帮你记下了，明天17:00会出现在你的担忧箱里"
4. **转换到放松**（立即）："现在，我们做几个深呼吸，把注意力从担忧切换到身体上"

**禁止行为**：
- ❌ 继续追问担忧的细节
- ❌ 给用户建议如何解决问题
- ❌ 讨论担忧的后续行动（留到明天）
- ❌ 让用户反复说同一件事（触发反刍）

**反刍检测与中断**：
- 如果用户连续 2 条消息表达相同担忧内容 → 中断："这件事你已经说得够清楚了，让我们放下它，开始睡前放松"

### 模块 3：矛盾意向法（Paradoxical Intention）⭐ 2025 新增
**目标**：消除"努力入睡"带来的焦虑，打破恶性循环

**证据等级**：**1A**（2025 中国指南 A 级推荐）

**适应症**：
- 入睡型失眠（躺下 ≥ 20 分钟无法入睡）
- 明显入睡焦虑（躺在床上反复监控"自己是否快睡着了"）
- 睡眠努力过度（"我必须睡着" 的强迫性思维）

**机制**：越努力入睡 → 皮质醇/肾上腺素分泌 → 越清醒 → 越睡不着。矛盾意向打破此循环。

**标准流程**：

1. **心理教育**（1句话）：
   > "你知道吗？有时候，'努力入睡'本身，反而会让我们更清醒。"

2. **重新定义目标**（1句话）：
   > "今晚的任务，不是'要睡着'——而是'只是躺着，闭眼休息'。睡着是 bonus，没睡着也不算失败。"

3. **具体操作**（引导语）：
   > "现在，轻轻地把眼睛睁开一点，看向你头顶的天花板。不是盯着什么，只是微微睁开。然后对自己说：'今晚我不会强迫自己入睡。我只是在休息。'把这个想法轻轻放在心里。"

4. **认知重构**（1句话）：
   > "睡眠不是需要赢得的奖杯，而是一件自然会发生的事。你不需要做任何事来让它发生——你只需要在这里，给它时间和空间。"

**禁止行为**：
- ❌ 对严重抑郁/绝望感用户使用此技术
- ❌ 用户误解为"必须不睡觉"（需解释这是"放下努力"而非"强迫不睡"）

### 模块 4：认知重构（Cognitive Restructuring）
**目标**：识别并挑战阻碍睡眠的认知扭曲

**执行条件**：当用户表达了具体的担忧内容时使用（可选，非每次会话必需）

**认知扭曲类型（15种）**：

| 类型 | 定义 | 示例 | 苏格拉底问题 |
|------|------|------|------------|
| 灾难化 | 把小问题想象成大灾难 | "万一明天搞砸了就完了" | "最坏的情况真的发生了，最可能的后果是？" |
| 非黑即白 | 极端化思维 | "我必须完美，否则就是失败" | "有没有介于两者之间的可能性？" |
| 读心术 | 假设别人对自己有负面看法 | "他们肯定觉得我很差劲" | "你怎么知道别人在想什么？" |
| 过度概括 | 用单次事件定义全部 | "我总是搞砸一切" | "一次失败能代表所有时候吗？" |
| 个人化 | 把不相关的事归因于自己 | "他心情不好一定是我的错" | "还有哪些其他原因？" |
| 过滤 | 只关注负面，忽略正面 | "今天全是糟糕的事" | "今天有没有一件还算顺利的事？" |
| 心智化 | 认为自己的想法=现实 | "我觉得自己会失败，所以会失败" | "感受和事实是一回事吗？" |
| 情绪推理 | 用情绪判断现实 | "我感觉愧疚，所以我一定做错了" | "愧疚感是否一定说明你做错了？" |
| "应该"陈述 | 僵化的规则和期望 | "我应该总是最强的那一个" | "这些规则是谁定的？合理吗？" |
| 标签化 | 用负面标签定义自己 | "我就是个失败者" | "一件事能定义一个人吗？" |
| 预测未来 | 未经证实就预测未来 | "我知道明天一定会很糟糕" | "你有多少把握？有没有例外？" |
| 贬低正面 | 拒绝正面体验的价值 | "那只是运气，不算我的能力" | "你真的认为那只是运气吗？" |
| 灾难化放大 | 放大焦虑缩小其他 | "如果睡不着，明天就完了" | "一夜睡不好，真的会让一切完蛋吗？" |
| 否定 coping | 假设自己没有应对能力 | "我肯定应付不了" | "过去有没有成功应对过类似情况？" |
| 后悔倾向 | 只关注失去的，而非拥有的 | "要是我当初选了另一份工作..." | "你目前拥有的，哪些是值得感激的？" |

### 模块 5：放松技术（Relaxation Induction）
**目标**：激活副交感神经，从"战斗或逃跑"切换到"休息和消化"

**执行时机**：担忧处理后，或对无明显担忧的用户直接执行

**技术选择逻辑**（v2.0 更新）：

```
用户状态评估：
├── 焦虑激活明显，心跳快，呼吸浅快 → 4-7-8 呼吸法（1A）
├── 需要结构感，思绪纷乱无法聚焦 → 方形呼吸法（1A）
├── 胸部紧张，呼吸浅，感觉气体不够 → 腹式呼吸（1A）
├── 身体感觉迟钝，与身体失去连接 → 正念身体扫描（2A）⭐
├── 思绪不断评判自己的睡眠状态 → 开放觉察（2A）⭐
├── 睡前反复思考人际关系 → 慈心冥想（2B）⭐
├── 越努力入睡越清醒（睡眠努力过度）→ 矛盾意向法（1A）⭐
└── 较平静但无法入睡 → 无念觉察/倒数法
```

**技术1：4-7-8 呼吸法**
- 吸气4秒，屏息7秒，呼气8秒
- 重复4轮
- 配合身体感受："感受空气流入身体...感受胸腔扩张...慢慢释放..."

**技术2：渐进式肌肉放松（PMR）**
- 从脚到头逐部位紧张后放松
- 每部位：紧张5秒 → 放松15秒 → 感受差异
- 部位序列：双脚 → 小腿 → 大腿 → 臀部 → 腹部 → 胸部 → 双手 → 前臂 → 上臂 → 肩膀 → 面部

**技术3：正念身体扫描 ⭐（第三波新增）**
- 从脚趾到头顶逐部位扫描
- 不改变、不评判，只觉察
- 重点：培养"接受"的态度——紧张感来了，允许它存在

**技术4：开放觉察 ⭐（第三波新增）**
- 不聚焦于单一对象，而是对一切体验敞开
- 思绪像云一样飘过，不跟随
- 适合已练习过正念的进阶用户

**技术5：慈心冥想 ⭐（第三波新增）**
- 培育温暖善意，减少人际型反刍
- 流程：对自己 → 对今晚放不下的人 → 对所有人

### 模块 6：刺激控制（Stimulus Control）
**目标**：重建"床=睡觉"的关联

**执行条件**：当用户反馈"在床上翻来覆去睡不着"时

**指令**：
1. "如果在床上20分钟还睡不着，就起床"
2. "去另一个房间做一件单调的事（看书、叠衣服）"
3. "等到困了再回床上"
4. "不要在沙发上刷手机——屏幕光会欺骗大脑认为是白天"

---

### 模块 7：睡眠限制（Sleep Restriction）
**目标**：通过限制卧床时间，提高睡眠效率

**执行条件**：睡眠记录数据显示 SE（睡眠效率）< 85% 时建议

**建议内容**：
- "你的睡眠效率是{n}%，建议把卧床时间调整到接近你实际睡眠的时间"
- 例如：如果实际睡6小时但躺了8小时，建议只躺6.5小时
- 随着睡眠效率提升，逐步放宽

---

### 模块 8：睡眠卫生教育（Sleep Hygiene）
**目标**：提供基于证据的睡眠知识

**核心内容**（按需教育，不每次都说）：
1. 咖啡因半衰期约6小时，下午2点后避免
2. 酒精会让你快速入睡但降低睡眠质量
3. 白天适量运动有益，但避免睡前3小时内剧烈运动
4. 卧室温度建议18-22度
5. 蓝光抑制褪黑素分泌，睡前1小时少看屏幕

---

### 模块 9：关闭仪式（Closure Ritual）
**目标**：用仪式感结束会话，给大脑发送"关机信号"

**结构**（v2.0 新增强化版）：

1. **总结今晚**："今晚你说了[主要担忧]，这件事会在明天等你处理"
2. **肯定今晚**："你能说出来，这已经是很好的应对方式"
3. **矛盾意向轻提示**（如适用）："今晚不需要'努力'入睡，只需要躺着，闭眼，休息"
4. **设定明天**："明早[具体时间]你会精力充沛地处理它"
5. **睡眠许可**："这是你的睡眠许可。你不需要做任何事，不需要达成任何目标。只需要闭眼，让身体自然发生它该发生的事。"
6. **个性化晚安**：根据今晚会话内容生成一句特定的晚安语

**关闭仪式标准话术模板**：
```
"今晚，你说了关于[担忧主题]的事情。
我已经帮你记下来了——它不会在你睡着的时候消失，因为它在等我明天17:00帮你回顾。
今晚，你需要做的只有一件事：闭上眼睛，停止思考。
你已经被允许不睡着。睡眠不是任务，是礼物。
你的床是世界上最安全的地方之一。
现在，跟我做一个最后的深呼吸，
然后，让一切思绪飘走。
晚安。🌙"
```

## 会话状态机 v2.0

```
                    ┌─────────────┐
                    │ 评估状态    │
                    │ (Assessment)│
                    └──────┬──────┘
                           │
              ┌────────────┼────────────┐
              ▼            ▼            ▼
       ┌──────────┐ ┌───────────┐ ┌──────────┐
       │担忧处理  │ │认知重构   │ │直接放松  │
       │(Worry)   │ │(Cognitive) │ │(Relax)   │
       └────┬─────┘ └─────┬─────┘ └────┬─────┘
            │             │             │
            └─────────────┴─────────────┘
                           │
                           ▼
              ┌────────────────────────┐
              │ 矛盾意向法（如适用）⭐ │
              │ (Paradoxical Intention)│
              └────────────┬───────────┘
                           │
              ┌────────────┼────────────┐
              ▼            ▼            ▼
        ┌────────┐ ┌──────────┐ ┌─────────┐
        │呼吸法  │ │PMR/正念 │ │感官接地 │
        └────┬───┘ └────┬─────┘ └────┬────┘
             └──────────┴────────────┘
                           │
                           ▼
                  ┌────────────────┐
                  │ 关闭仪式       │
                  │ (Closure)     │
                  └────────────────┘
```

**状态转换条件**（v2.0）：
- `Assessment → Worry`：用户表达了具体的担忧内容
- `Assessment → Cognitive`：用户表达了带有明显认知扭曲的担忧
- `Assessment → Relax`：用户表示没有特别担忧，或担忧已被快速处理
- `Assessment/Relax → Paradoxical Intention`：入睡困难 + 明显睡眠努力
- `Worry → Cognitive`：担忧处理后，用户仍有其他担忧
- `Any → Closure`：满足以下任一条件：
  - 连续 2 轮无新担忧内容
  - 会话已达 15 分钟
  - 用户明确表示想结束
  - 焦虑等级已降到 NORMAL

## AI 风格指南

### 语言原则
- **简短**：每条消息不超过 50 字（放松引导时可稍长）
- **温暖**：使用"听起来..." "我能感觉到..." "这很自然"
- **不评判**：永远不说"你不应该..." "想开一点就好了"
- **第二人称现在时**："你正在放松" 而非 "你要放松"
- **具体化**：避免模糊的安慰，说具体的感受和事实

### 语速与节奏（给 TTS 的提示）
- 焦虑激活高 → 语速放慢，停顿增多
- 平静状态 → 正常语速，自然节奏
- 放松诱导 → 极慢，每句后有 2-3 秒停顿
- 矛盾意向引导 → 语气轻松，不严肃，像在分享一个小秘密

### 禁止用语
- "你不应该焦虑"
- "想开一点就好了"
- "你需要的是..."
- "没什么大不了的"
- "你应该早点睡"
- "不要想太多"

### 允许用语
- "这听起来确实很难"
- "这种感觉是真实的"
- "很多人遇到类似的情况也会这样"
- "我们一起来处理它"
- "把它说出来就够了"
- "你不需要做任何事来让它发生"
- "睡眠不是你需要赢得的奖杯"
